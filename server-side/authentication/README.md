# Authentication

[OWASP Authentication Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Authentication_Cheat_Sheet.html)

Authentication vulnerabilities allow attackers to bypass login mechanisms, impersonate users, or gain unauthorized access to sensitive functionality.

## Common impact

- Account takeover through credential theft, brute-force, or session hijacking
- Bypass of multi-factor authentication (2FA) mechanism
- Unauthorized access to user data and privileged operations
- Exposure of valid usernames, enabling further targeted attacks
